Integrating DITA into NetBeans

Integration with NetBeans

The NetBeans editor provides many powerful features for editing many files and format. In particular it support xml very well.

There are two ways to integrate DITA with NetBeans.

The first must be done before this DITA module can process the documentation. Select Tools > Options > DITA and define the installed path for the DITA Open Toolkit.

The second involves registering the DITA xml definition with NetBeans so that the editor can provide hints and auto-completion of the tags. Select Tools > DTDs and XML Schemas follow these steps.
  1. Click Add Catalogue
  2. In the dialog click Browse
  3. Browse to the root directory of the DITA Open Toolkit.
  4. Select the file catalog-dita.xml
  5. Click OK.
  6. Now open a dita file and auto-completion will be enabled.
